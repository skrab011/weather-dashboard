# Weather Dashboard

Personal weather-consolidation PWA for two Colorado locations (Silverthorne home, Frisco office). See `CLAUDE.md` for the locked spec and build plan.

## Local dev

```bash
npm install
npm run dev
```

## Stack

- Frontend: Vite + TypeScript (PWA shell added in workstream 2)
- Backend: Vercel serverless functions (`/api`)
- Hosting: Vercel free tier
